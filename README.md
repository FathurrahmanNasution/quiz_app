## Deskripsi Aplikasi
Aplikasi kuis interaktif berbasis Flutter yang menguji pengetahuan pengguna tentang KMNZ (Virtual YouTuber music unit). Aplikasi ini memiliki fitur:
- Input nama pengguna sebelum memulai kuis
- Informasi trivia tentang KMNZ
- 10 pertanyaan pilihan ganda
- Sistem penilaian otomatis
- Tampilan hasil akhir dengan statistik
- Desain responsif untuk berbagai ukuran layar
- State management menggunakan Provider
- Navigasi antar halaman dengan go_router

**Teknologi yang digunakan:**
- Flutter SDK (>=3.0.0)
- Provider (State Management)
- go_router (Navigation)
- Custom Font: Poppins
